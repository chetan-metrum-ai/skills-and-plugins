# Metrum AI video visual system

Use this system for every video created with this skill. The source of truth is
the [Metrum AI website](https://www.metrum.ai/). Refresh the site and its assets
before a production when possible; do not infer a different brand from a third
party's visual asset.

## Required foundation

- Use the official white logo from
  `https://www.metrum.ai/assets/img/metrum_logo_white_new.png` on dark fields.
  Preserve its aspect ratio, do not redraw it, and keep clear space around it.
- Use a near-black or black background, white primary copy, and restrained gray
  secondary copy. Use thin technical rules, subtle grid/topology cues, square or
  minimally rounded UI elements, and deliberate empty space.
- Use the Metrum gradient in this order when an accent is needed:
  `#FF3132 → #FE005F → #EE0089 → #CC28AF → #9948CB → #465CDA`.
  Do not introduce unrelated accent colors.

## Type roles

- Use **Geist** for display headlines; favor light or regular weights with tight
  tracking.
- Use **Poppins** for body copy and explanatory labels.
- Use **Geist Mono** for small uppercase navigation, metadata, timestamps,
  technical labels, and calls to action; use modest letter spacing.

The site currently serves these fonts from its own static assets. Load them from
the official site or use licensed local equivalents; do not silently substitute a
different visual family when a Metrum asset is available.

## Motion and composition

Keep motion restrained and purposeful: gentle opacity fades, short line reveals,
and data-flow motion suit the interface aesthetic. Avoid bouncy motion, cartoon
illustration, glossy 3D effects, rounded consumer-app cards, or generic template
styles. Keep the official logo visible on title and closing frames; use it on
other frames only when it does not compete with the content.

## Source record

This specification was derived from the public Metrum AI homepage: black theme,
white logo, Geist/Poppins/Geist Mono type roles, and the listed red-to-blue
gradient. Recheck the source for changes before work that requires exact current
brand parity.
