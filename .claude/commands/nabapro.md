# nabapro

Use this skill when the user needs to generate, edit, or iterate on images with Google's Gemini image-generation API (Nano Banana Pro). Trigger on requests for product visuals, brand/campaign imagery, text-in-image rendering, reference-based editing, style transfer, video-to-image, multi-turn refinement, or any Gemini image workflow. Applies Metrum AI branding by default unless the user provides another brand.

This command delegates to the agent skill at `../skills/nabapro/`.
